## Overview

Development of a rental management web application with bank integration (bank account scanning) and online payment.

## Key Features

- **Project Management**: Developed tools for streamlined project tracking, task assignment, and progress monitoring, ensuring smooth workflows.
- **Direct Customer Communication**: Implemented features for real-time communication between teams and end customers, enhancing transparency and client satisfaction.
- **File Management & Annotations**: Built a robust system for managing files, enabling seamless annotations and timecode-based reviews for efficient collaboration.
- **File Playback & Timecode Reviewing**: Integrated a media player with advanced timecode functionality, allowing precise review and feedback on multimedia content.

## Technologies Used

- **React.js & Next.js**: For fast, dynammic, responsive and server-rendered web app.
- **Supabase**: For backend services, including authentication, database management, and real-time functionality.
- **KonvaJs**: For interactive canvas-based features, enabling advanced annotations and graphical editing.
- **Video.Js**: For customizable video playback, supporting a wide range of formats and timecode-based reviewing.

## Challenges and Learnings

One of the main challenges was ensuring seamless integration between file management, annotations, and real-time communication features. By adopting modular architecture and leveraging modern development tools, I was able to create a cohesive system that meets diverse user needs. Collaborating closely with stakeholders to refine requirements and deliver a user-friendly solution was a key learning experience.

## Outcome

The ERP program has significantly improved operational efficiency, enabling teams to manage projects, review files, and communicate with clients in a unified platform. It has become an essential tool for delivering high-quality results while maintaining strong client relationships.
